# Registros, Estadísticas y Monitoreo

## Registros de ejecución

```sh
sudo lukso network log execution -f
```

Puedes cerrar los registros pulsando `ctrl+c`.

## Registros de consenso

```
sudo lukso network log consensus -f
```

Puedes cerrar los registros pulsando `ctrl+c`.
